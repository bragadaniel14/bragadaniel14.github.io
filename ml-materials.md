<section class="section-block">
  <h1>Gentle ML Introductions and Tutorials</h1>
  <p>
    A curated list of online materials I enjoyed for learning machine learning concepts with clear intuition. You can use this as a refresher if you already learned it in the past or as a primer if you are about to get into it. At the end of the day, practicing problem with an actual book or class is still the best way to "make it stick", but I have enjoyed listening to some of these on my free time.
  </p>
</section>

    <article class="card">
      <h2>Classes/Tutorials</h2>
      <ul class="resource-list">
        <li>
          <details class="resource-item">
            <summary><a href="https://ocw.mit.edu/courses/18-03-differential-equations-spring-2010/video_galleries/video-lectures/" target="_blank" rel="noopener">MIT 18.03 - Differential Equations</a></summary>
            <p>
              I never took a full differential equations class. This lecture series gave me practical patterns for ODEs and stronger intuition for Fourier and Laplace transforms, which has been useful when reading AI work on dynamical systems and physical models.
            </p>
          </details>
        </li>
        <li>
          <details class="resource-item">
            <summary><a href="https://cs229.stanford.edu/materials.html" target="_blank" rel="noopener">Stanford CS229 Materials</a></summary>
            <p>Classic notes for fundamentals.</p>
          </details>
        </li>
        <li>
          <details class="resource-item">
            <summary><a href="https://www.deeplearning.ai/courses/deep-learning-specialization/" target="_blank" rel="noopener">deeplearning.ai Deep Learning Specialization</a></summary>
            <p>
              If you enjoy theory-heavy classes, this is a good complement because it focuses on turning ideas into working code while still explaining important architectural details.
            </p>
          </details>
        </li>
        <li>
          <details class="resource-item">
            <summary><a href="https://uvadlc-notebooks.readthedocs.io/" target="_blank" rel="noopener">UVA Deep Learning Course Notebooks</a></summary>
            <p>I used this to understand JAX better. It is well written, mostly self-contained, and has clear PyTorch equivalents.</p>
          </details>
        </li>
      </ul>
