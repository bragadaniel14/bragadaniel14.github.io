title: "A Useful ML Learning Loop"
date: 2026-03-03
categories: [tech]
---
One simple loop that helps me learn new ML topics:

1. Build intuition from one visual/video explanation.
2. Read one technical source with equations.
3. Reimplement the core idea in a tiny notebook.
4. Summarize what failed and what finally worked.

I learn faster when I avoid collecting too many resources before coding.
